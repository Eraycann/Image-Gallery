# Image Gallery

![Project Demo](readme.gif)

---

## Overview

This project showcases a stylish and modern image gallery created with HTML, CSS, and JavaScript. The gallery provides an intuitive way for users to navigate through a collection of captivating images.

---

## How to Use üöÄ

1. **Clone the Repository:**

    ```bash
    git clone https://github.com/Eraycann/Image-Gallery.git
    ```

2. **Open the Project:**

    Open the `index.html` file in your preferred web browser.

3. **Navigate Through Images:**

    - Use the mouse wheel for smooth vertical scrolling through the gallery.
    - Click the "Next" button to elegantly transition to the next set of images.
    - Click the "Back" button to gracefully move back to the previous set of images.

## Project Structure

- `images/`: Contains image files used in the gallery.
- `style.css`: Stylesheet defining the gallery layout.
- `index.html`: HTML file outlining the gallery structure and including the necessary scripts.
